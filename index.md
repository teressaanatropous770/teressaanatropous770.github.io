## 🔍 Understanding the Features

LGS offers several tools to assist with image interpretation.

### Region Localization
The core feature places bounding boxes around detected anomalies in your images. Each box correlates to a specific sentence in the report. This visual link helps verify the AI findings against your clinical knowledge.

### Automated Report Generation
The model writes a summary based on the visual data. You can edit the text before you save the report. This feature allows for constant human oversight.

### Model Selection
You can choose different versions of the underlying language model. Some models prioritize speed for quick scans, while others look for very small patterns in the data.
